# SensorThing
ESP8266 + DHT22 RoomSensor

## Features
- Runs with 4x AA batteries (21µA Standby current => 1+ years runtime)
- Measures temperature, relative humidity and battery voltage
- Calculates dew point and absolute humidity
- Pushes data via HTTP POST to server
